# Changelog

## v0.2.1 — 달력에서 일정 추가
- 달력의 **날짜를 클릭하면 그 날짜를 마감일로 한 작업을 바로 추가**(구글 캘린더식). 추가된 작업은 즉시 달력에 표시됩니다.
- 기존 퀵 캡처 모달을 재사용하되, 클릭한 날짜를 기한으로 미리 채우고 모달에서 수정 가능. 날짜 셀에 hover 시 `+` 표시.
- `toDateInput`을 로컬 날짜 기준으로 수정(시간대 +9/+14 등에서 하루 밀리던 문제 제거) — 클릭한 날짜와 달력 표시 날짜가 모든 시간대에서 일치하도록 보정.
- **확인**: 달력에서 빈 날짜 클릭 → 제목 입력 후 추가 → 해당 날짜 칸에 작업 표시. 작업 칩 클릭은 추가가 아닌 해당 작업으로 이동.

## v0.2 — UX 개선

기존 구조(Electron + React + TS + Zustand + Tailwind/shadcn + better-sqlite3, 로컬 우선, IPC 브리지)와
soft delete · `updated_at` 규칙을 유지한 채 아래 UX를 추가했다.

### P0-1 키보드 우선 조작
- 전역: `⌘/Ctrl+N` 퀵 캡처, `⌘/Ctrl+P` 검색 팔레트, `?` 도움말, `Esc` 닫기.
- 리스트: `j/k` 이동, `Enter` 열기, `Space/x` 완료, `e` 제목 편집, `0–3` 우선순위.
- 키맵을 `src/renderer/src/shortcuts.ts` 한 곳에서 관리하고 도움말 오버레이가 이를 그대로 렌더.
- 입력창 포커스 시 단일키 단축키 비활성화(`isTypingTarget`).
- **확인**: 리스트에서 `j/k`로 하이라이트 이동 → `Space` 완료 토글 → `e` 제목 편집 → `1/2/3/0` 우선순위 변경. `?`로 도움말. 입력 중에는 단축키 미작동.

### P0-2 인라인 편집
- 제목: 더블클릭 또는 `e` → Enter 저장 / Esc 취소.
- 기한: 행에서 인라인 날짜 피커로 즉시 변경(없으면 `+` 아이콘).
- 우선순위: 배지 클릭 순환 또는 숫자키.
- 모든 변경은 낙관적 업데이트 + `updated_at` 갱신.
- **확인**: 작업 행에서 우선순위 배지 클릭 시 0→1→2→3 순환, 기한 배지 클릭 시 날짜 선택 즉시 반영.

### P1-1 드래그앤드롭 정렬 (@dnd-kit/core + sortable)
- 리스트: 행 드래그(그립 핸들) → `sort_order` 갱신.
- 칸반: 컬럼 내 정렬 + 컬럼 간 이동(status 변경) → 변경분만 일괄 업데이트.
- 카테고리: 루트 카테고리 순서 변경 → `sort_order` 갱신.
- `sort_order` 전략: 데이터가 적어 **정수 재인덱스 + 트랜잭션 일괄 업데이트**를 택함(`task.repo.reorder` 주석 참고). 변경된 행만 `update()`를 거쳐 `updated_at`/`completed_at` 규칙 유지.
- 리스트/칸반 정렬을 위해 목록 정렬 기준을 `sort_order` 우선(완료는 하단)으로 변경.
- **확인**: 리스트에서 그립으로 순서 변경, 칸반에서 카드를 다른 컬럼으로 끌면 상태 변경, 같은 컬럼 내 재정렬. 카테고리 패널에서 루트 순서 변경 후 재시작해도 유지.

### P1-2 스마트 뷰 "오늘 / 이번 주" (cross-workspace)
- 사이드바 상단에 데스크와 별개의 가상 뷰.
- 모든 데스크를 가로질러 `due_date <= 오늘/이번주말`(지난 작업 포함) 집계. `task.repo.listUpcoming`.
- 각 작업에 사업/카테고리 컬러 점 + 라벨. 지난 작업은 상단 "지남" 섹션으로 강조.
- done 토글·인라인 편집 동작. `Enter`(또는 제목 클릭) 시 해당 작업의 데스크로 이동해 펼침.
- **확인**: 여러 데스크에 마감일 작업 생성 → "오늘"에 오늘+지난 작업이 컨텍스트 라벨과 함께 표시, 지난 작업이 상단. "이번 주"는 이번 주말까지 포함.

### P1-3 검색 / 빠른 이동 팔레트
- `⌘/Ctrl+P` 중앙 모달. 작업(제목/note)·카테고리·데스크 통합 LIKE 검색.
- `↑↓` 선택 + `Enter` 이동. 검색 쿼리는 `search.repo`로 분리(추후 FTS 전환 대비).
- **확인**: `⌘P` → 키워드 입력 → 결과를 화살표로 선택 후 Enter → 해당 작업/카테고리/데스크로 이동.

### P2-1 OS 네이티브 알림
- 메인 프로세스에서 오늘 마감·미완료 작업을 시작 시 + 60초 주기 + 창 포커스 시 점검(`src/main/notifications.ts`).
- 세션 메모리 `Set`으로 같은 작업 중복 알림 방지(과설계 방지: 컬럼 미사용).
- 알림 클릭 → 창 포커스 + 렌더러로 `navigate-to-task` 전송 → 해당 작업으로 이동.
- **확인**: 오늘 마감 미완료 작업이 있으면 실행 직후 알림. 클릭 시 앱이 떠오르고 해당 작업이 펼쳐짐.

### P2-2 다크 / 라이트 / 시스템 테마
- Tailwind `class` 전략 + CSS 변수 토큰(라이트/다크). 사이드바 토글(해/달/모니터).
- `localStorage`에 선택 영속, `system`은 OS 변경을 실시간 추종.
- **확인**: 테마 토글 시 즉시 전환, 재시작 후 유지, `system` 선택 시 OS 다크모드 토글에 따라 변경.
